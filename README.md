## 📋 What is GPT-Voyager?

GPT-Voyager is a browser extension designed to improve your use of ChatGPT. It helps you organize conversations, use ready-made prompts, extract formulas and diagrams, and save your work in popular formats like Markdown and HTML. It also supports local backups so you don’t lose your data.

You can use it with Google Chrome or other Chromium-based browsers on Windows.

## 🛠 Troubleshooting

If the extension does not work as expected:

- Check that Chrome is up to date.  
- Confirm developer mode is enabled in `chrome://extensions/`.  
- Make sure the extension has permission to read and change data on the ChatGPT site.  
- Restart Chrome after installing or updating the extension.  
- Disable other extensions that might conflict with GPT-Voyager.  
- Clear your browser cache if the UI looks broken.  
- If extracting formulas or exporting fails, try reloading the ChatGPT page.
