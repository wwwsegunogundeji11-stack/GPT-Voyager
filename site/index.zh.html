FAQ

安装前常见问题

会把我的聊天数据上传到外部服务器吗？

不会。当前版本为本地优先，默认仅在浏览器本地存储扩展数据。

如何手动加载扩展？

构建完成后打开 chrome://extensions，开启开发者模式并加载 dist/。

可以跨设备迁移数据吗？

可以。使用 JSON 导出/导入迁移索引、提示词、设置等数据。

支持哪些浏览器？

支持 Manifest V3 的 Chromium 浏览器，主要是 Chrome 与 Edge。

INSTALL GUIDE

两种安装路径

路径 A：Release 包安装（推荐）

  1. 打开 Latest Release。
  2. 下载发布 ZIP 资产并解压。
  3. 打开 chrome://extensions，开启开发者模式。
  4. 点击“加载已解压的扩展程序”，选择解压后的扩展目录。

路径 B：源码构建安装

  1. 下载 源码 ZIP 或直接克隆仓库。
  2. 在项目根目录运行 npm install。
  3. 运行 npm run build 生成 dist/。
  4. 打开 chrome://extensions 并加载 dist/。
