# Chrome Web Store 上架文案（可直接粘贴）

## 1. 名称
`GPT Voyager`

## 2. Summary（短描述，建议 <= 132 字符）
`为 ChatGPT 网页提供会话索引、提示词库、公式/图表工作台与本地备份的效率扩展。`

## 3. Description（长描述）
GPT Voyager 是一个面向 ChatGPT 网页端的本地优先效率扩展，帮助你在不离开对话页面的前提下完成会话管理、内容复用与技术信息整理。

核心能力：
- 会话索引与搜索：自动采集可见会话，支持关键词、文件夹、标签、星标筛选
- 会话组织与批量操作：支持分类、备注、批量修改与撤销
- 提示词库：支持模板管理、变量填充、预设复用、批量导出
- 公式工作台：提取公式并复制 LaTeX / Word 可粘贴 MathML
- Mermaid 工作台：识别图表代码、预览、收藏、导出（源码/SVG/HTML）
- 导出与备份：支持会话 Markdown/HTML 导出与本地 JSON 备份恢复
- 轻量侧边栏：支持宽度调整、快捷键、简约界面与高频操作优化

适用人群：
- 重度 ChatGPT 用户
- 需要归档对话与知识沉淀的工程师/研究者/写作者
- 需要频繁复用提示词与技术内容（公式/图表）的用户

## 4. 隐私声明建议（可用于商店隐私字段）
- 扩展默认本地存储数据，不上传用户对话内容到开发者服务器
- 扩展仅在 chatgpt.com / chat.openai.com 页面生效
- 用户可自行导出/删除本地数据
