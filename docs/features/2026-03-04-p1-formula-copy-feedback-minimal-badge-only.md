## 2. 概述
根据反馈继续收敛交互视觉：页面点击公式复制时，反馈从“描边 + 胶囊”进一步精简为“仅胶囊提示”。

## 3. 需求映射
- 对应 PRD 章节：
  - FR-07 设置（复制反馈体验）
- 覆盖的用户场景：
  - 用户希望反馈更简洁，不要在公式区域出现额外框线。
- 范围说明（In scope / Out of scope）：
  - In scope：
    - 删除页面复制反馈中的描边框与光晕
    - 保留 Word / LaTeX 文案区分
  - Out of scope：
    - 新增反馈样式配置项

## 4. 设计
- 关键设计决策：
  - 保留局部位置感知（提示在公式附近），但不再绘制覆盖框。
  - 继续使用文档坐标的绝对定位，滚动时定位行为自然。

## 5. 实现
- 主要变更文件：
  - `src/content/App.tsx`
- 核心逻辑：
  - `showPageFormulaCopyFeedback`：
    - 移除 `ring` DOM 创建与动画。
    - 只渲染 `badge` 胶囊提示并保留淡入淡出动效。
    - 保留旧节点清理，避免快速连续点击时提示叠加。

## 6. 验证
- 自动化测试：
  - `npm run typecheck`：通过
  - `npm run build`：通过
- 手工验证步骤：

  2. 确认只出现“已复制 Word 公式 / 已复制 LaTeX”胶囊提示。
  3. 确认不再出现框住公式的描边层。
