# FEAT-047：侧边栏 UI 重构（极简白色专业风）

## 1. 元信息
- 功能 ID：FEAT-047
- 功能名称：侧边栏 UI 重构（极简白色专业风）
- 日期：2026-03-05
- 作者：Codex
- 分支：main

## 2. 概述
基于 `ui-ux-pro-max` 的设计规则，对 GPT Voyager 侧边栏进行一次可读性与层级导向的重构：
- 视觉方向：极简白色专业风（去米黄、去杂色、保留绿色为主强调色）。
- 结构方向：增强顶部信息层级（品牌区 + 功能分区 + 当前视图说明）。
- 交互方向：统一控件高度、焦点态与反馈节奏，提升对齐一致性。

## 3. 需求映射
- 对应需求：
  - “对齐更好、层级更清晰”
  - “极简白色专业风”
- 技术策略：
  - 设计令牌先行（色板/边框/阴影/间距）
  - 组件级统一（按钮、输入框、卡片、标签）
  - 可访问性补强（focus-visible、reduced-motion）

## 4. Skill 使用说明
- 使用技能：`ui-ux-pro-max`
- 实施方式：
  1. 读取技能文档中的优先级规则（可访问性、交互、布局、配色）。
  2. 尝试运行本地安装 skill 的 `--design-system` 脚本时，发现该 skill 在本机为路径指针文件（`scripts` 非目录）。
  3. 从其公开仓库拉取同源脚本并执行：
     - `search.py "chatgpt extension sidebar minimal white professional" --design-system -p "GPT Voyager Panel"`
  4. 将输出结论映射为当前扩展样式重构（白底、高对比、轻阴影、单层重点）。

## 5. 实现
- 主要变更文件：
  - `src/content/App.tsx`
  - `src/content/index.tsx`
  - `docs/FEATURE_LOG.md`

### 5.1 结构层级优化（App）
- 新增顶部结构容器：`gv-topbar`
- 新增当前视图说明区：`gv-view-intro`
  - 显示 `kicker / title / description`，按当前 tab 动态切换
- 保持功能不变，仅重组信息层级以降低视觉噪音

### 5.2 视觉系统重构（CSS）
- 重置设计令牌：
  - 主背景改为白/浅灰梯度
  - 线框与文字对比增强
  - 强调色统一绿色（避免多色冲突）
- 面板与导航：
  - 顶部 sticky 区域（品牌 + 分区 + 视图简介）
  - 导航按钮改为 2 列栅格，提升对齐与可读性
- 组件统一：
  - 按钮、输入、下拉统一 34px 高度基线
  - 卡片圆角、边框、hover 阴影统一
  - 星标、标签、提示气泡统一中性基调
- 列表与卡片：
  - 会话、时间线、公式、Mermaid、提示词、设置、指南等卡片统一白底 + 轻边框

### 5.3 可访问性与动效
- 新增全局 `:focus-visible` 高可见焦点环
- 新增 `prefers-reduced-motion` 适配，降低动画时长

## 6. 验证
- 自动化：
  - `npm run typecheck` 通过
- 手工检查项：
  1. 打开侧边栏后，顶部三段结构（品牌/导航/视图说明）清晰可见。
  2. 各控件高度与边界对齐一致，无明显“跳高/错位”。
  3. 各模块卡片在白色主题下对比清晰，层级关系稳定。

## 7. 风险与回滚
- 已知风险：
  - 视觉统一后，个别用户可能偏好更强的彩色区分。
- 回滚策略：
  - 回退 `src/content/App.tsx` 与 `src/content/index.tsx` 到重构前版本。
