# FEAT-038：页面公式复制反馈样式与定位修复

## 1. 元信息
- 功能 ID：FEAT-038
- 功能名称：页面公式复制反馈样式与定位修复
- 日期：2026-03-04
- 作者：Codex
- 分支：N/A

## 2. 概述
针对“点击公式复制后绿色框观感粗糙、滚轮滚动时反馈看起来跟着鼠标移动”的问题，完成反馈样式与定位机制修复：
- 反馈视觉改为轻量描边 + 小胶囊文案，降低突兀感。
- 反馈定位从 `fixed` 改为基于文档坐标的 `absolute`，避免滚动时出现“悬浮跟随鼠标”的错觉。

## 3. 需求映射
- 对应 PRD 章节：
  - FR-07 设置（复制反馈体验相关）
- 覆盖的用户场景：
  - 用户点击页面公式复制后，希望反馈清晰但不刺眼。
  - 页面滚动时反馈应与目标内容保持自然一致，不应像悬浮窗一样固定在视口。
- 范围说明（In scope / Out of scope）：
  - In scope：
    - 页面公式复制反馈动效样式优化
    - 反馈定位逻辑修正
  - Out of scope：
    - 新增全局通知系统
    - 用户自定义反馈颜色与动画参数

## 4. 设计
- 关键设计决策：
  - 点击时先清理旧反馈节点，避免多次点击叠加。
  - 保留 `Word` / `LaTeX` 两种反馈文案区分。
  - 使用文档坐标（`scrollX/scrollY + getBoundingClientRect`）定位反馈元素。
- 取舍说明：
  - 继续保留局部反馈而非全局 toast，能更直接关联到被点击公式。

## 5. 实现
- 主要变更文件：
  - `src/content/App.tsx`
- 核心逻辑：
  - 重写 `showPageFormulaCopyFeedback`：
    - 新增旧节点清理：
      - `gpt-voyager-formula-copy-ring`
      - `gpt-voyager-formula-copy-badge`
    - 反馈元素 `position` 从 `fixed` 改为 `absolute`。
    - 大面积渐变底色改为细边框与轻微 glow。
    - 胶囊提示字体、阴影、动画节奏统一收敛。

- 自动化测试：
